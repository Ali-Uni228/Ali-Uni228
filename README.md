#### 🛒 DeathLeade-Store

<div align="center">

[![DeathLeade-Store](https://github-readme-stats.vercel.app/api/pin/?username=Ali-Uni228&repo=DeathLeade-Store&theme=nord&border_color=f59e0b&title_color=f59e0b&text_color=c9d1d9&bg_color=0d1117)](https://github.com/Ali-Uni228/DeathLeade-Store)

</div>

A full-stack Minecraft gaming network storefront designed for automated package delivery using transaction verification and TCP RCON server communication. The frontend is publicly deployed; backend services are intended to run on a dedicated server environment.

| Layer | Technology |
|---|---|
| Backend | Node.js, Express.js |
| Database | MongoDB, SQLite |
| Game Server Integration | TCP RCON |
| Frontend | HTML/CSS, JavaScript |

**🔴 Live:** [deathleade-store.netlify.app](https://deathleade-store.netlify.app/) &nbsp;•&nbsp; **🔗 Code:** [github.com/Ali-Uni228/DeathLeade-Store](https://github.com/Ali-Uni228/DeathLeade-Store)
